#### Tools and Frameworks Used
1. Arduino uno 
2. GSM Module SIM900A 
3. Turbidity sensor SKU:SEN 0189
4. Waterproof temperature sensor DS18B20
5. 16X2 LCD Display
6. Arduino IDE for coding  
7. Thingspeak for uploading data to internet
